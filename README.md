🏨 Hospitality Data Analytics Dashboard (Power BI)

--------------------------------------------------------------------------------------------------------------
📌 Project Overview
--------------------------------------------------------------------------------------------------------------

This project focuses on Hospitality Domain Analytics using Power BI.
The objective is to analyze hotel performance metrics help stakeholders make data-driven business decisions.

--------------------------------------------------------------------------------------------------------------
This dashboard provides insights into:
--------------------------------------------------------------------------------------------------------------

Hotel performance
Revenue trends
Occupancy analysis
Booking patterns
Weekly performance tracking

--------------------------------------------------------------------------------------------------------------
🎯 Business Problem
--------------------------------------------------------------------------------------------------------------
Hospitality businesses need to track multiple KPIs to optimize:

Room utilization
Revenue generation
Booking trends
Performance comparison across properties
